Lenguajes de programación y marcado

Información sobre algunos lenguajes de programación

Python

Python es un lenguaje de programación con una sintaxis bastante sencilla, por lo que es famoso por ser el lenguaje favorito de personas que inician en la programación.

Este lenguaje es muy versátil y puede ser utilizado para el desarrolo de IA como para el backend de una página web.


Ir a Python.org

HTML

HTML no es un lenguaje de programación sino de marcado, lo que significa que el navegador web es el encargado de ejecutar lo que se escribe.

Este lenguaje se utiliza para la estructura de las páginas web.


Ir a Documentación HTML

CSS

CSS es un lenguaje de marcado que sirve junto con HTML para el diseño de las páginas web.

Pero no solo web, también se pueden diseñar videojuegos básicos.
